# MintOS

本实验参考OrangeS代码，调用其chapter11/a/代码。

## 使用说明

调用时直接输入`$ sh bash.sh`，因为多需要make command文件。

遇到

```
Device:[HD]
Messaget ata0-0t could not open hard drive image file '80m,img'
```

是因为bochs没有正常退出，把目录下的`80m.img.lock`删除后再调用bash.sh即可。
